# Cadastro de Usuários


## Instalação

* Baixar o NPM e o Node

* Abrir o projeto em uma IDE de preferência ou no terminal do sistema operacional

  * `npm install`

  * `npm run dev`
  
* Aplicação por padrão, estará disponível em `localhost:3000`
